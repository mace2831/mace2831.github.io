Software Engineering and Design

In this enhancement I converted the AnimalShelter project from CS-340 from a JupyterDash Python project to a MEAN stack web application. I refactored the Python project to fit the MEAN stack and created the backend logic as and Express application adding API endpoints and controllers to drive the backend logic. I created a Mongo database to hold the various animal information and display it as a list with clickable rows to take the user to the selected animal's detail page. The detail page displays the information of each animal and provides an interactive map showing the location of the animal. Databases

In this artifact I further refined the AnimalShelter project two from CS-340 by completing the single page administrator application that limits database manipulation to only authorized users. I added this artifact into my ePortfolio to show my ability to create a secure web application that protects data in the database while allowing users to safely view the information. This enhancement clearly showcases my security mindset by anticipating security breaches and mitigating vulnerabilites. The Trie data structure from the second enhancement that is used for the autocomplete function prevents any sort of injection attack to the database as the interaction with the database is done without user input. When taking in the username from the user Mongoose sanitize is used to sanitize the input to remove threats from injection attacks. On the administration page the buttons that allow the user to manipulate the data in the database only become visible when a valid user and JWT certificate are verified. The course outcomes that align with this enhancement are 3 and 5 and demonstrated my ability to design and evaluate computing solutions that sove a given problem as well as show my security mindset by anticipating exploits and mitigate those vulnerabilities through the implementation of data sanitization and user authorization and authentication. This process was easier than adding the autocomplete functionality because I have created a single page application administrator page before. Adding in the data sanitization through Mongoose sanitize was an easy process, but a good learning experience.
